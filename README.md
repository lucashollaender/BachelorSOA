This repository contains the source and development code for the Bachelor Project: A Spatial Operator Algebra Formulation for Efficient Flexible Multibody Dynamics and Applications by Markus, Lucas and Jeppe. The Development Code folder contains the preliminary scripts used for developing, testing, verification and performance analysis, while SOALIB contains a self-developed Spatial Operator Algebra library, which is used extensively throughout the code. These folders contain raw development and testing code and are not intended for general use or exploration. Instead, the reader is invited to the SOA Implementation Code folder, which contains the final developed implementations. To explore the capabilities of the Python-based rigid and flexible simulations, the reader is encouraged to run the Test\<...>.py scripts located within the Rigid SOA and Flexible SOA subdirectories.
